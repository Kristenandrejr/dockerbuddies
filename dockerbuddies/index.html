This simple guide introduces Docker in a beginner-friendly way using Python and HTML.

1. What is Docker?

Docker is a tool that helps developers package applications into containers ‚Äî lightweight environments that include everything the app needs to run.

2. Common Docker Commands

  • docker pull hello-world - Download a sample image
  • docker run hello-world - Run a container from the image
  • docker ps - List running containers
  • docker images - View all downloaded images

3. Example Dockerfile

# Start from a lightweight Python image
FROM python:3.9-slim

# Set working directory
WORKDIR /app

# Copy files
COPY . .

# Install dependencies
RUN pip install -r requirements.txt

# Run the app
CMD ["python", "app.py"]
